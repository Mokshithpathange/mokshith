
1. git branch 				- to list the branch
2. git branch <branch-name> 		- to create the branch
3. git checkout -b <branch-name>	- to create and switch the branch
4. git switch <branch-name>		- to switch the branch 
5. git push origin <branch-name>	- to push the the branch to git hub 
6. git pull origin <branch-name> 	- to pull the branch from git hub
7. git branch -r			- to list the branch from remotely

Branching Commands

git branch
